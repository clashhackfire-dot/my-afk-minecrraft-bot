# ⛏️ Mineflayer AFK Bot Dashboard

A 24/7 Minecraft AFK bot with a beautiful web dashboard built with Node.js, Express, WebSocket, and Mineflayer.

## ✨ Features

- 🎮 Live Dashboard - Real-time bot monitoring and control
- 🔄 Auto-Restart - Automatically restarts if kicked
- 💬 Chat Integration - Send messages from the dashboard
- 👥 Player List - View all online players with health
- ❤️ Health & Food Display - Track bot's vital stats
- 🔐 Password Protection - Secure dashboard access
- 💾 Persistent Settings - Saved to local storage and config file
- 🖥️ Live Console - Real-time log viewer
- 🎨 Sleek UI - Beautiful dark theme with status indicators
